# 数组

## 什么是数组

```markdown
数组：在`连续的`内存空间中，存在一组`相同类型`的元素
```

## 时间复杂度

```markdown
1. 访问 Access O(1)	
	访问是根据索引找到值
	
2. 搜索 Search O(N)	
	遍历找值，与索引无关
	
3. 插入 Insert O(N)	
	数组是在连续的内存空间中，所以插入时很可能要移动元素。[1, 2, 3] -> [4, 1, 2, 3]这是最坏的情况，需要将1,2,3分别向后移动一位。如果没有足够空间了，那么需要重新开辟一块空间。
	
4. 删除 Delete O(N)
	[1, 2, 3] -> [2, 3]	需要将2,3分别向前移动一步。
```

## 特点

```markdown
- 适合`读多写少`
```


### LeetCode485

最大连续1的个数



```java
public static int findMax(int[] nums){
    if(nums == null || nums.length() == 0){
        return 0;
    }
    int count = 0;
    int result = 0;
    for(int i = 0; i < nums.length(); i++){
        if(nums[i] == 1){
        	count += 1;
        }
        else{
            result = Math.max(result, count);
            count = 0;
        }
    }
    return result
}
```

### LeetCode283

给定一个数组 nums ，编写一个函数将所有0移动到数组的末尾，同时保持非零元素的相对顺序。

示例：

```
输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
```

说明：

1. 必须在原数组上操作，不能拷贝额外的数组。
2. 尽量减少操作次数。

```java
public static int[] leetCode283(int[] nums){
    int index = 0;
    for(int i = 0; i < nums.length(), i++){
        if(nums[i] != 0){
        	nums[index] = nums[i];
            index += 1;
        }
    }
    for(int index; i < nums.length(); index++){
        nums[index] = 0;
    }
    return nums;
}
```

> 其实遍历一遍数组，找到不为0的元素，把他放到第index个位置，index+1，继续遍历。剩下的就全部是0了！
