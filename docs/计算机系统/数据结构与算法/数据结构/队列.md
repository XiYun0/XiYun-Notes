# 队列

## 特点

先进先出



## 队列常用操作

### 创建队列

```java
// 使用链表的原因在于，队列都是一边插入一边删除，而这正是链表的优势
Queue<Integer> queue = new LinkedList<>();
```

### 添加元素

```java
queue.add(1);
queue.add(2);
queue.add(3);
```

### 获取元素

```java
int temp1 = queue.peek();
// 1
```

### 删除元素

```java
int temp2 = queue.poll();
```

### 判断队列是否为空

```java
// O(1)
queue.isEmpty();
```

### 队列长度

```java
// O(1)
queue.size();
```

### 遍历队列

```java
// O(1)
while(!queue.isEmpty()){
    int temp = queue.poll();
}
```
