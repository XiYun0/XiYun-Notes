# 集合

## 特点

无序，不重复。

## 集合的常用操作

### 创建集合

```java
Set<Integer> set = new HashSet<>();
```

### 添加元素

```java
// O(1)
set.add(10);
set.add(3);
set.add(5);
set.add(2);
set.add(2);
// [10, 3, 5, 2]  不可重复，所以2只能有1个
```

### 搜索元素

```java
// O(1)
set.contains(2);
```

### 删除集合

```java
// O(1)
set.remove(2);
```

### 长度

```java
set.size();
```

## 练习

### LeetCode217存在重复元素



```java
public static boolean leeCode_217(int[] nums){
    if(nums == null || nums.length() == 0){
        return false;
    }
    Set<Integer> set = new HashSet<>();
    for(int i = 0; i <= nums.length(); i++){
        set.add(nums[i]);
    }
    if(set.size() == nums.length()){
        return false;
    }else{
        return true;
    }
}
```
