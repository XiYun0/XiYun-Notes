# 队列

## 特点

```
先进后出
```



应用：浏览器后退



## 栈常用操作

### 创建栈stack

```java
Stack<Integer> stack = new Stack<>();
```

### 添加元素

```java
// O(1)
// 压入栈中
stack.push(1);
stack.push(2);
stack.push(3);
```

### 获取栈顶元素

```java
stack.peek();
```

### 删除栈顶元素

```java
int temp = stack.pop();
```

### 栈的大小

```java
stack.size();
```

### 栈是否为空

```java
stack.isEmpty();
```

### 栈的遍历

```java
while(!stack.isEmpty()){
    stack.pop();
}
```
