# Linux命令

## 查询和卸载

### rpm

查询

```
rpm -aq | grep 
```

卸载

```
rpm -e 服务
```

### 进程、端口

```
lsof -i:端口
netstat -pan | grep 端口
```

> 这两个都可以得到进程

```
ps -ef | grep 进程
```

> 找到具体的程序

### 文件

模糊查询文件

```
find / -name *grafana*
```

> 不加/就是在当前目录下搜索

模糊查询目录

```
find / -name "grafana"
```

> 可以不带 ""
