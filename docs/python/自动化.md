## subprocess

### 打开软件

```python
import subprocess

a = input('请输入：')

process = subprocess.Popen(['start', a], shell=True)
process.wait()
```

### 使用命令

```python
import subprocess

process = subprocess.Popen('ipconfig', shell=True)
process.wait()
```
